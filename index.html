Dúvidas Frequentes:

Não consigo redefinir minha senha

    Use o recurso de troca de senha e aguarde o e-mail de Redefinição. Caso não chegue em 24 horas, por favor, use o botão "Solicitações Canvas" acima.

Minhas disciplinas NÃO aparecem no Canvas

    Conferir sua Grade Curricular no Portal do Aluno. A disciplina deve estar com status de "MATRICULADO", caso não esteja, procurar a Central do Aluno para conferir possíveis pendências.

    Caso esteja com o status "MATRICULADO", por favor, aguardar 72 horas. Após esse período, use o botão "Solicitações Canvas" acima.

Não consigo configurar meu aplicativo Canvas para receber notificações

    Na página 04 do Guia Canvas Afya para Estudante, seguindo o passo a passo, você consegue configurar o app mobile para receber notificações de: vencimento de tarefas, avisos, atividades novas e etc... As notificações também estão disponíveis para serem recebidas via e-mail. É importante lembrar que é de responsabilidade do aluno verificar seu Canvas constantemente.

    Atenção ao Aplicativo! Use somente para receber atualizações de tarefas, testes, fóruns, avisos e notificações via notificação no celular. Problemas no envio de atividades, tarefas, testes ou de qualquer outra natureza através do aplicativo é de inteira responsabilidade do aluno. Não prestamos apoio para o aplicativo mobile e nem nos responsabilizamos por erros que possam acontecer no app do seu celular. Funcionalidades podem não funcionar como esperadas comparados com a visualização do Canvas em um navegador web totalmente suportado.
